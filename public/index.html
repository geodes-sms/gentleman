Gentleman

Gentleman is a lightweight web modelling tool that allows you to
create and manipulate concept-based models with projections.

  • Docs Guide and Tutorial
  • FAQ Questions and Answers
Play with Gentleman Gentleman in action

Features

  • Easy integration with any web application
  • Compatible with Ecore model
  • Support textual and tabular notations and multimedia content
  • Support multiple projections for a concept
  • Easy to switch the projections used against a model
  • Projections are defined with specialized UI elements

Examples

  • Mindmap

    Organize your thoughts as a topic tree: central → main → sub → sub… and assign a marker to any topic for further organization.

  • Traffic Light

    Configure a traffic light system as a state machine of lights and modes composed of lights.

  • ReLiS

    Configure a ReLiS project with ease from the screening phase to the report phase depending on your need.
